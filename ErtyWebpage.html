The JPEG Image

The term "JPEG" is an acronym for the
Joint Photographic Experts Group, which
created the standard. The MIME media
type for JPEG is image/jpeg, except in
older Internet Explorer versions, which
provides a MIME type of image/pjpeg
when uploading JPEG images. JPEG files
usually have a filename extension of .jpg
or .jpeg.

JPEG/JFIF supports a maximum image
size of 65,635 x 65,535 pixels hence up
to 4 gigapixels (for an aspect ration of 1:1).
